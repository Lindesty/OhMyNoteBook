## 项目概述

这是一个面向有静态强类型面向对象编程经验（Java/C#/C++）的开发者的 TypeScript 快速入门教程。包含完整的 Markdown 文档和可运行的示例代码。

## 编写规范

1. **目标受众**：有 Java/C#/C++ 经验的开发者
2. **内容风格**：简洁、对比式、实用为主
3. **代码示例**：可运行、有注释、有输出
4. **章节结构**：
   - 概述
   - 语法说明
   - 代码示例
   - 与 Java/C# 对比
   - 最佳实践

## 与 Java/C# 主要差异

| 特性 | Java/C# | TypeScript |
|------|---------|------------|
| 类型系统 | 名义类型 | 结构化类型 |
| 方法重载 | 多个实现 | 多签名 + 单实现 |
| 泛型 | 运行时存在 | 编译时擦除 |
| 空值处理 | 可空引用类型 | null/undefined/可选属性 |
| 枚举 | 类型安全 | 生成运行时代码 |
| 装饰器/注解 | 语言特性 | 实验性特性 |

## 注意事项

- 所有示例代码使用 ESM 模块（`"type": "module"`）
- 使用 tsx 运行 TypeScript，无需预编译
- 章节顺序已优化，请勿随意调整
- 文档在 `docs/` 目录，示例在 `examples/` 目录

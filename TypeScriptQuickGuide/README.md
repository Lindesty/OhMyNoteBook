## 与 Java/C# 主要差异

| 特性 | Java/C# | TypeScript |
|------|---------|------------|
| 类型系统 | 名义类型 | 结构化类型 |
| 方法重载 | 多个实现 | 多签名 + 单实现 |
| 泛型 | 运行时存在 | 编译时擦除 |
| 空值处理 | 可空引用类型 | null/undefined/可选属性 |
| 枚举 | 类型安全 | 生成运行时代码 |
| 装饰器/注解 | 语言特性 | 实验性特性 |

## 学习建议

1. **先看第 00 章**，了解 TypeScript 与你熟悉语言的主要差异
2. **按顺序学习**，章节安排遵循由浅入深的原则
3. **运行示例代码**，每个章节都有对应的可运行示例
4. **对比学习**，思考 TypeScript 与 Java/C# 的设计差异

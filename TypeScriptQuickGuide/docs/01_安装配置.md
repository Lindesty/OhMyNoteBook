# 安装配置

本章节介绍如何安装和配置 TypeScript 开发环境。对于有 Java/C# 经验的开发者，这类似于安装 JDK 或 .NET SDK。

## 安装 Node.js

TypeScript 运行在 Node.js 环境中，首先需要安装 Node.js。

### 下载安装

从 [Node.js 官网](https://nodejs.org/) 下载并安装 LTS 版本。

### 使用版本管理器（推荐）

**Windows**: 使用 [nvm-windows](https://github.com/coreybutler/nvm-windows)

```powershell
# 安装 nvm-windows 后
nvm install 20
nvm use 20
```

**macOS/Linux**: 使用 [nvm](https://github.com/nvm-sh/nvm)

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 20
nvm use 20
```

### 验证安装

```bash
node -v   # 显示 Node.js 版本
npm -v    # 显示 npm 版本
```

## 安装 TypeScript

### 全局安装

```bash
npm install -g typescript
```

### 验证安装

```bash
tsc -v
# Version 5.x.x
```

### 项目本地安装（推荐）

```bash
# 在项目目录中
npm install --save-dev typescript
```

## 运行 TypeScript

### 方式一：编译后运行

```bash
# 编译
tsc hello.ts

# 运行编译后的 JavaScript
node hello.js
```

### 方式二：使用 ts-node

```bash
npm install -g ts-node

# 直接运行 TypeScript
ts-node hello.ts
```

### 方式三：使用 tsx（推荐）

tsx 是更快的 TypeScript 执行器：

```bash
npm install -g tsx

# 直接运行
tsx hello.ts

# 监听模式
tsx watch hello.ts
```

## 创建项目

### 简单单文件项目

```
my-project/
├── hello.ts
└── package.json
```

```bash
# 初始化项目
npm init -y

# 安装 TypeScript
npm install --save-dev typescript tsx
```

### 配置 tsconfig.json

```bash
# 生成配置文件
npx tsc --init
```

基本配置示例：

```json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "ESNext",
    "moduleResolution": "bundler",
    "strict": true,
    "outDir": "./dist",
    "rootDir": "./src"
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules"]
}
```

### 完整项目结构

```
my-project/
├── src/
│   ├── index.ts
│   └── utils.ts
├── dist/              # 编译输出目录
├── node_modules/
├── package.json
└── tsconfig.json
```

## package.json 脚本

```json
{
  "scripts": {
    "build": "tsc",
    "start": "tsx src/index.ts",
    "dev": "tsx watch src/index.ts",
    "typecheck": "tsc --noEmit"
  }
}
```

运行：

```bash
npm run build     # 编译
npm start         # 运行
npm run dev       # 开发模式（监听变化）
npm run typecheck # 仅类型检查
```

## IDE 配置

### VS Code（推荐）

1. 安装 TypeScript 扩展（通常内置）
2. 使用 `Ctrl+Space` 获取智能提示
3. 使用 `F12` 跳转到定义
4. 使用 `F2` 重命名符号

### WebStorm / IntelliJ IDEA

内置 TypeScript 支持，无需额外配置。

### 配置保存时自动格式化

在 `.vscode/settings.json` 中：

```json
{
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "typescript.preferences.importModuleSpecifier": "relative"
}
```

## 常见问题

### 找不到模块

确保安装了类型定义：

```bash
npm install --save-dev @types/node
```

### 编译错误

检查 `tsconfig.json` 配置：

```bash
# 详细输出
npx tsc --listEmittedFiles
```

### 版本冲突

使用项目本地的 TypeScript：

```bash
npx tsc  # 使用项目本地版本
```

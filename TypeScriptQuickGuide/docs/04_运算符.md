# 运算符

TypeScript 继承了 JavaScript 的所有运算符，并增加了类型相关的运算。对于 Java/C# 开发者，大部分运算符都很熟悉，但有一些重要差异。

## 算术运算符

```ts
const a = 10, b = 3;

console.log(a + b);  // 13 加法
console.log(a - b);  // 7  减法
console.log(a * b);  // 30 乘法
console.log(a / b);  // 3.333... 除法（总是浮点数）
console.log(a % b);  // 1  取余
console.log(a ** b); // 1000 幂运算（ES2016）

// 自增/自减
let x = 5;
console.log(++x); // 6 前置
console.log(x++); // 6 后置
console.log(x);   // 7
```

### 与 Java/C# 的差异

```ts
// JavaScript/TypeScript 除法总是浮点数
console.log(10 / 3);  // 3.333...
// Java: 10 / 3 = 3 (整数除法)

// 幂运算使用 **
console.log(2 ** 10); // 1024
// Java: Math.pow(2, 10)
// C#: Math.Pow(2, 10) 或 2.Pow(10)
```

## 比较运算符

```ts
const a = 5, b = "5";

// 宽松比较（会进行类型转换）
console.log(a == b);   // true
console.log(a != b);   // false

// 严格比较（推荐，不进行类型转换）
console.log(a === b);  // false
console.log(a !== b);  // true

// 大小比较
console.log(10 > 5);   // true
console.log(10 >= 10); // true
console.log(5 < 10);   // true
console.log(5 <= 5);   // true
```

### == vs ===（重要差异）

```ts
// ❌ 避免 == 和 !=
console.log(0 == false);    // true
console.log("" == 0);       // true
console.log(null == undefined); // true

// ✅ 推荐 === 和 !==
console.log(0 === false);   // false
console.log("" === 0);      // false
console.log(null === undefined); // false
```

## 逻辑运算符

```ts
const a = true, b = false;

console.log(a && b);  // false 逻辑与
console.log(a || b);  // true  逻辑或
console.log(!a);      // false 逻辑非

// 短路求值
const name = null;
const displayName = name || "Unknown"; // "Unknown"
const value = true && "success";       // "success"
```

### 空值合并运算符（??）

```ts
// 只有 null 和 undefined 触发默认值
const a = null ?? "default";      // "default"
const b = undefined ?? "default"; // "default"
const c = "" ?? "default";        // ""（空字符串不触发）
const d = 0 ?? "default";         // 0（零不触发）

// 与 || 的区别
const e = 0 || "default";   // "default"（0 是 falsy）
const f = 0 ?? "default";   // 0
```

### 可选链运算符（?.）

```ts
interface User {
  name: string;
  address?: {
    city: string;
  };
}

const user: User = { name: "Tom" };

// 可选链
const city = user.address?.city; // undefined（不会报错）

// 函数可选链
const fn = user.address?.toString?.();

// 数组可选链
const items: string[] | undefined = undefined;
const first = items?.[0]; // undefined
```

## 位运算符

```ts
const a = 5, b = 3; // 二进制: 101, 011

console.log(a & b);   // 1   按位与
console.log(a | b);   // 7   按位或
console.log(a ^ b);   // 6   按位异或
console.log(~a);      // -6  按位非
console.log(a << 1);  // 10  左移
console.log(a >> 1);  // 2   右移
console.log(a >>> 1); // 2   无符号右移
```

## 赋值运算符

```ts
let a = 10;

a += 5;   // a = a + 5
a -= 3;   // a = a - 3
a *= 2;   // a = a * 2
a /= 4;   // a = a / 4
a %= 3;   // a = a % 3
a **= 2;  // a = a ** 2
a &= 1;   // a = a & 1
a |= 2;   // a = a | 2
a ^= 3;   // a = a ^ 3
a <<= 1;  // a = a << 1
a >>= 1;  // a = a >> 1

// 逻辑赋值
let b = null;
b ??= "default"; // b = b ?? "default"
let c = false;
c ||= true;      // c = c || true
let d = true;
d &&= false;     // d = d && false
```

## 三元运算符

```ts
const age = 20;
const status = age >= 18 ? "成年" : "未成年";

// 嵌套（避免过度嵌套）
const grade = score >= 90 ? "A" :
              score >= 80 ? "B" :
              score >= 70 ? "C" : "D";
```

## 类型运算符

### typeof

```ts
typeof "hello";      // "string"
typeof 123;          // "number"
typeof true;         // "boolean"
typeof undefined;    // "undefined"
typeof null;         // "object"（历史遗留问题）
typeof {};           // "object"
typeof [];           // "object"
typeof function(){}; // "function"
typeof Symbol();     // "symbol"
typeof 123n;         // "bigint"
```

### instanceof

```ts
const arr = [1, 2, 3];
console.log(arr instanceof Array);   // true
console.log(arr instanceof Object);  // true

const date = new Date();
console.log(date instanceof Date);   // true
```

### in

```ts
const user = { name: "Tom", age: 30 };

console.log("name" in user);  // true
console.log("email" in user); // false
```

## 展开运算符

```ts
// 数组展开
const arr1 = [1, 2, 3];
const arr2 = [...arr1, 4, 5]; // [1, 2, 3, 4, 5]

// 对象展开
const obj1 = { a: 1, b: 2 };
const obj2 = { ...obj1, c: 3 }; // { a: 1, b: 2, c: 3 }

// 函数参数展开
function sum(...numbers: number[]): number {
  return numbers.reduce((a, b) => a + b, 0);
}
```

## 运算符优先级

从高到低：

| 优先级 | 运算符 |
|--------|--------|
| 1 | `()` `.` `?.` `[]` `()`（调用） |
| 2 | `!` `~` `+` `-` `typeof` `void` `delete` |
| 3 | `**` |
| 4 | `*` `/` `%` |
| 5 | `+` `-` |
| 6 | `<<` `>>` `>>>` |
| 7 | `<` `<=` `>` `>=` `in` `instanceof` |
| 8 | `==` `!=` `===` `!==` |
| 9 | `&` |
| 10 | `^` |
| 11 | `\|` |
| 12 | `&&` |
| 13 | `\|\|` `??` |
| 14 | `? :` |
| 15 | `=` `+=` `-=` 等 |

## 与 Java/C# 的差异总结

| 运算符 | Java | C# | TypeScript |
|--------|------|-----|------------|
| 相等比较 | `equals()` | `Equals()` | `===` |
| 幂运算 | `Math.pow()` | `Math.Pow()` | `**` |
| 空值合并 | `Optional.orElse()` | `??` | `??` |
| 可选链 | `Optional.map()` | `?.` | `?.` |
| 除法 | 整数除法 | 整数除法 | 浮点除法 |
| 类型检查 | `instanceof` | `is` | `typeof`/`instanceof` |

# 流程控制

TypeScript 的流程控制语句与 JavaScript 相同，对于 Java/C# 开发者非常熟悉。但有一些语法差异需要注意。

## 条件语句

### if-else

```ts
const score = 85;

if (score >= 90) {
  console.log("优秀");
} else if (score >= 80) {
  console.log("良好");
} else if (score >= 60) {
  console.log("及格");
} else {
  console.log("不及格");
}
```

### switch

```ts
const day = "Monday";

switch (day) {
  case "Monday":
    console.log("星期一");
    break;
  case "Tuesday":
    console.log("星期二");
    break;
  case "Wednesday":
  case "Thursday":
    console.log("星期三或四");
    break;
  default:
    console.log("其他");
}
```

### switch 表达式模式（使用对象模拟）

```ts
const status = "pending";

const label = {
  pending: "待处理",
  approved: "已通过",
  rejected: "已拒绝"
}[status] ?? "未知状态";
```

## 循环语句

### for 循环

```ts
// 传统 for 循环
for (let i = 0; i < 5; i++) {
  console.log(i);
}

// for...of（遍历值）
const colors = ["red", "green", "blue"];
for (const color of colors) {
  console.log(color);
}

// for...in（遍历键/索引）
const user = { name: "Tom", age: 30 };
for (const key in user) {
  console.log(`${key}: ${user[key as keyof typeof user]}`);
}

// for await...of（异步迭代）
async function processUrls(urls: string[]) {
  for await (const url of urls) {
    // 异步处理
  }
}
```

### while 循环

```ts
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}
```

### do-while 循环

```ts
let j = 0;
do {
  console.log(j);
  j++;
} while (j < 5);
```

## 循环控制

### break

跳出整个循环：

```ts
for (let i = 0; i < 10; i++) {
  if (i === 5) break;
  console.log(i); // 0, 1, 2, 3, 4
}
```

### continue

跳过当前迭代：

```ts
for (let i = 0; i < 5; i++) {
  if (i === 2) continue;
  console.log(i); // 0, 1, 3, 4
}
```

### 标签语句

```ts
outer: for (let i = 0; i < 3; i++) {
  for (let j = 0; j < 3; j++) {
    if (i === 1 && j === 1) {
      break outer; // 跳出外层循环
    }
    console.log(`(${i}, ${j})`);
  }
}
```

## 数组方法替代循环

在 TypeScript/JavaScript 中，推荐使用数组方法代替传统循环：

### forEach

```ts
const numbers = [1, 2, 3, 4, 5];

numbers.forEach((num, index) => {
  console.log(`索引 ${index}: ${num}`);
});
```

### map

转换数组元素：

```ts
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(n => n * 2);
// [2, 4, 6, 8, 10]
```

### filter

过滤数组元素：

```ts
const numbers = [1, 2, 3, 4, 5];
const evens = numbers.filter(n => n % 2 === 0);
// [2, 4]
```

### find 和 findIndex

查找元素：

```ts
const users = [
  { id: 1, name: "Tom" },
  { id: 2, name: "Jerry" }
];

const user = users.find(u => u.id === 2);
// { id: 2, name: "Jerry" }

const index = users.findIndex(u => u.name === "Tom");
// 0
```

### some 和 every

检查条件：

```ts
const numbers = [1, 2, 3, 4, 5];

const hasEven = numbers.some(n => n % 2 === 0); // true
const allPositive = numbers.every(n => n > 0);  // true
```

### reduce

累积计算：

```ts
const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce((acc, n) => acc + n, 0);
// 15
```

## 与 Java/C# 的对比

| 特性 | Java | C# | TypeScript |
|------|------|-----|------------|
| for-each | `for (Type item : collection)` | `foreach (Type item in collection)` | `for (const item of collection)` |
| switch 表达式 | ✅ (Java 14+) | ✅ | ❌ 使用对象模拟 |
| 模式匹配 | ✅ (Java 16+) | ✅ | ❌ 使用类型守卫 |
| LINQ | ❌ | ✅ | 使用数组方法链 |
| Stream API | ✅ | ❌ | 使用数组方法 |

**Java 示例**：
```java
List<String> names = users.stream()
    .filter(u -> u.getAge() > 18)
    .map(User::getName)
    .collect(Collectors.toList());
```

**TypeScript 等价**：
```ts
const names = users
    .filter(u => u.age > 18)
    .map(u => u.name);
```

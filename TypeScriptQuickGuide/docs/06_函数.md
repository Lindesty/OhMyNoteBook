# 函数

TypeScript 函数在 JavaScript 基础上增加了类型系统，支持函数重载、泛型和多种参数模式。

## 函数声明

### 函数声明语法

```ts
// 基本语法
function add(a: number, b: number): number {
  return a + b;
}

// 函数表达式
const subtract = function(a: number, b: number): number {
  return a - b;
};

// 箭头函数
const multiply = (a: number, b: number): number => a * b;
```

### 类型别名

```ts
// 函数类型别名
type MathOperation = (a: number, b: number) => number;

const divide: MathOperation = (a, b) => a / b;
```

### 接口定义

```ts
interface SearchFunc {
  (source: string, subString: string): boolean;
}

const search: SearchFunc = (source, sub) => source.includes(sub);
```

## 参数

### 可选参数

```ts
function greet(name: string, greeting?: string): string {
  return `${greeting ?? "Hello"}, ${name}!`;
}

greet("Tom");              // "Hello, Tom!"
greet("Tom", "Hi");        // "Hi, Tom!"
```

### 默认参数

```ts
function greet(name: string, greeting: string = "Hello"): string {
  return `${greeting}, ${name}!`;
}

greet("Tom");              // "Hello, Tom!"
greet("Tom", "Hi");        // "Hi, Tom!"
```

### 剩余参数

```ts
function sum(...numbers: number[]): number {
  return numbers.reduce((acc, n) => acc + n, 0);
}

sum(1, 2, 3);       // 6
sum(1, 2, 3, 4, 5); // 15
```

### 解构参数

```ts
interface Config {
  url: string;
  method?: "GET" | "POST";
  timeout?: number;
}

function request({ url, method = "GET", timeout = 5000 }: Config): void {
  console.log(`${method} ${url} (timeout: ${timeout}ms)`);
}

request({ url: "/api/users" });
request({ url: "/api/posts", method: "POST", timeout: 3000 });
```

## 返回值

### 返回类型注解

```ts
function getUser(id: number): User | null {
  // ...
}

// void 表示无返回值
function log(message: string): void {
  console.log(message);
}

// never 表示永不返回
function throwError(message: string): never {
  throw new Error(message);
}
```

### 返回类型推断

```ts
// TypeScript 自动推断返回类型
function add(a: number, b: number) { // 返回 number
  return a + b;
}

function getUser(id: number) { // 返回 User | undefined
  return users.find(u => u.id === id);
}
```

## 函数重载

TypeScript 支持函数重载，但只能有一个实现：

```ts
// 重载签名
function format(value: string): string;
function format(value: number): string;
function format(value: Date): string;

// 实现
function format(value: string | number | Date): string {
  if (typeof value === "string") {
    return `String: ${value}`;
  }
  if (typeof value === "number") {
    return `Number: ${value.toFixed(2)}`;
  }
  return `Date: ${value.toISOString()}`;
}

format("hello");   // "String: hello"
format(3.14159);   // "Number: 3.14"
format(new Date()); // "Date: 2024-..."
```

## this 类型

```ts
interface User {
  name: string;
  greet(this: User): void;
}

const user: User = {
  name: "Tom",
  greet() {
    console.log(`Hello, I'm ${this.name}`);
  }
};
```

## 泛型函数

```ts
// 泛型函数
function identity<T>(arg: T): T {
  return arg;
}

identity<string>("hello");
identity(123); // 类型推断

// 多类型参数
function pair<K, V>(key: K, value: V): [K, V] {
  return [key, value];
}

// 泛型约束
function getProperty<T, K extends keyof T>(obj: T, key: K): T[K] {
  return obj[key];
}
```

## 高阶函数

```ts
// 函数作为参数
function execute(fn: () => void): void {
  fn();
}

execute(() => console.log("Hello"));

// 函数作为返回值
function createMultiplier(factor: number): (x: number) => number {
  return (x) => x * factor;
}

const double = createMultiplier(2);
double(5); // 10

// 柯里化
function curry<A, B, C>(fn: (a: A, b: B) => C): (a: A) => (b: B) => C {
  return (a: A) => (b: B) => fn(a, b);
}
```

## 箭头函数 vs 普通函数

```ts
class Counter {
  count = 0;

  // 普通函数：this 指向调用者
  increment() {
    this.count++;
  }

  // 箭头函数：this 指向定义时的上下文
  decrement = () => {
    this.count--;
  };
}

const counter = new Counter();
const { increment, decrement } = counter;

increment(); // ❌ 错误：this 丢失
decrement(); // ✅ 正确：箭头函数绑定了 this
```

## 与 Java/C# 的对比

| 特性 | Java | C# | TypeScript |
|------|------|-----|------------|
| Lambda | `(x) -> x * 2` | `x => x * 2` | `x => x * 2` |
| 可选参数 | 重载 | 默认值 | `?` 和默认值 |
| 命名参数 | 不支持 | 支持 | 解构参数 |
| 重载 | 真正重载 | 真正重载 | 类型签名+单实现 |
| 泛型方法 | `<T> T method()` | `T Method<T>()` | `function method<T>()` |
| 高阶函数 | Stream API | LINQ | 直接支持 |

**Java 函数式接口**：
```java
@FunctionalInterface
interface MathOperation {
    int apply(int a, int b);
}

MathOperation add = (a, b) -> a + b;
```

**TypeScript 函数类型**：
```ts
type MathOperation = (a: number, b: number) => number;
const add: MathOperation = (a, b) => a + b;
```

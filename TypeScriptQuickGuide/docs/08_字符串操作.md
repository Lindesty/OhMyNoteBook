# 字符串操作

TypeScript 继承了 JavaScript 的字符串操作方法。对于 Java/C# 开发者，大部分方法都很熟悉，但语法略有不同。

## 字符串基础

### 创建字符串

```ts
// 单引号、双引号、反引号
const single = 'Hello';
const double = "World";
const template = `Hello, ${double}!`; // 模板字符串

// 多行字符串
const multiline = `
  第一行
  第二行
  第三行
`;

// 原始字符串
const raw = String.raw`C:\Users\Documents`;
```

### 字符串属性

```ts
const str = "Hello";
console.log(str.length); // 5

// 访问字符
console.log(str[0]);     // "H"
console.log(str.charAt(0)); // "H"
console.log(str.charCodeAt(0)); // 72 (Unicode 编码)
```

## 常用方法

### 查找

```ts
const str = "Hello, World!";

// 查找子串位置
str.indexOf("World");      // 7
str.lastIndexOf("o");      // 8
str.search(/World/);       // 7

// 是否包含
str.includes("World");     // true
str.startsWith("Hello");   // true
str.endsWith("!");         // true
```

### 提取

```ts
const str = "Hello, World!";

// 截取
str.slice(0, 5);          // "Hello"
str.slice(7);             // "World!"
str.substring(0, 5);      // "Hello"

// 分割字符
str.split(", ");          // ["Hello", "World!"]
str.split("");            // ["H", "e", "l", "l", "o", ...]

// 提取字符
str.charAt(0);            // "H"
str[0];                   // "H"
```

### 修改

```ts
const str = "Hello, World!";

// 大小写转换
str.toUpperCase();        // "HELLO, WORLD!"
str.toLowerCase();        // "hello, world!"
str.toLocaleUpperCase();  // 本地化大写
str.toLocaleLowerCase();  // 本地化小写

// 去除空白
"  hello  ".trim();       // "hello"
"  hello  ".trimStart();  // "hello  "
"  hello  ".trimEnd();    // "  hello"

// 重复
"ab".repeat(3);           // "ababab"

// 替换
str.replace("World", "TypeScript"); // "Hello, TypeScript!"
str.replaceAll("o", "0"); // "Hell0, W0rld!"
str.replace(/o/g, "0");   // 使用正则替换全部

// 填充
"5".padStart(3, "0");     // "005"
"5".padEnd(3, "0");       // "500"
```

### 连接

```ts
// concat 方法
"Hello".concat(", ", "World"); // "Hello, World"

// 展开运算符
const parts = ["Hello", "World"];
const result = parts.join(" "); // "Hello World"

// 模板字符串
const name = "Tom";
const greeting = `Hello, ${name}!`;
```

## 模板字符串

### 基本用法

```ts
const name = "Tom";
const age = 30;

// 变量插值
const str = `Name: ${name}, Age: ${age}`;

// 表达式
const sum = `1 + 2 = ${1 + 2}`;

// 调用函数
const upper = `Hello, ${name.toUpperCase()}!`;

// 嵌套模板
const isAdult = age >= 18;
const status = `Status: ${isAdult ? "Adult" : "Minor"}`;
```

### 标签模板

```ts
function highlight(strings: TemplateStringsArray, ...values: any[]): string {
  return strings.reduce((result, str, i) => {
    const value = values[i] !== undefined ? `<strong>${values[i]}</strong>` : "";
    return result + str + value;
  }, "");
}

const name = "Tom";
const age = 30;
const result = highlight`Name: ${name}, Age: ${age}`;
// "Name: <strong>Tom</strong>, Age: <strong>30</strong>"
```

## 类型转换

### 转换为字符串

```ts
// String()
String(123);        // "123"
String(true);       // "true"
String(null);       // "null"
String(undefined);  // "undefined"
String([1, 2, 3]);  // "1,2,3"

// toString()
(123).toString();   // "123"
(3.14).toString();  // "3.14"
(255).toString(16); // "ff" (十六进制)

// 模板字符串
`${123}`;           // "123"

// JSON 序列化
JSON.stringify({ name: "Tom" }); // '{"name":"Tom"}'
```

### 从字符串转换

```ts
// parseInt / parseFloat
parseInt("123");           // 123
parseInt("123px");         // 123
parseInt("ff", 16);        // 255
parseFloat("3.14");        // 3.14

// Number()
Number("123");             // 123
Number("3.14");            // 3.14
Number("hello");           // NaN

// 一元加号
+"123";                    // 123
```

## Unicode 支持

```ts
const str = "𝌆"; // Unicode 字符

// 长度（注意代理对）
str.length;              // 2（实际是 1 个字符）
[...str].length;         // 1（使用扩展运算符）

// Unicode 转义
"\u0041";                // "A"
"\u{1F600}";             // "😀"

// 获取 Unicode 码点
"😀".codePointAt(0);     // 128512
String.fromCodePoint(128512); // "😀"
```

## 与 Java/C# 的对比

| 操作 | Java | C# | TypeScript |
|------|------|-----|------------|
| 字符串插值 | `"Hello " + name` | `$"Hello {name}"` | `` `Hello ${name}` `` |
| 多行字符串 | 不支持 | `@""` | `` `...` `` |
| 判断是否为空 | `str.isEmpty()` | `string.IsNullOrEmpty()` | `str.length === 0` |
| 分割 | `str.split(",")` | `str.Split(',')` | `str.split(",")` |
| 连接 | `String.join(", ", arr)` | `string.Join(", ", arr)` | `arr.join(", ")` |
| 格式化 | `String.format()` | `string.Format()` | 模板字符串 |

**Java 示例**：
```java
String name = "Tom";
String greeting = String.format("Hello, %s!", name);
```

**TypeScript 等价**：
```ts
const name = "Tom";
const greeting = `Hello, ${name}!`;
```

# 序列

TypeScript 中的序列类型包括数组、元组、Map、Set 等。对于 Java/C# 开发者，这些类型非常类似，但语法和用法有差异。

## 数组（Array）

### 创建数组

```ts
// 字面量
const numbers: number[] = [1, 2, 3, 4, 5];
const names: string[] = ["Tom", "Jerry"];

// 泛型语法
const items: Array<number> = [1, 2, 3];

// 构造函数
const empty: number[] = new Array(5); // 长度为5的空数组
const filled: number[] = new Array(5).fill(0); // [0, 0, 0, 0, 0]

// Array.from
const fromSet = Array.from(new Set([1, 2, 3]));
const fromString = Array.from("hello"); // ["h", "e", "l", "l", "o"]
```

### 数组操作

```ts
const arr = [1, 2, 3];

// 添加元素
arr.push(4);        // 末尾添加
arr.unshift(0);     // 开头添加
const newArr = [...arr, 5]; // 不修改原数组

// 删除元素
arr.pop();          // 删除末尾
arr.shift();        // 删除开头
const filtered = arr.filter(x => x !== 2); // 不修改原数组

// 查找
arr.indexOf(2);     // 索引
arr.includes(2);    // 是否包含
arr.find(x => x > 1); // 查找元素
arr.findIndex(x => x > 1); // 查找索引

// 切片
arr.slice(1, 3);    // [2, 3]
arr.slice(-2);      // 最后两个元素
```

### 数组方法

```ts
const numbers = [1, 2, 3, 4, 5];

// 遍历
numbers.forEach((n, i) => console.log(n));

// 映射
const doubled = numbers.map(n => n * 2);

// 过滤
const evens = numbers.filter(n => n % 2 === 0);

// 归约
const sum = numbers.reduce((acc, n) => acc + n, 0);

// 检查
const hasEven = numbers.some(n => n % 2 === 0);
const allPositive = numbers.every(n => n > 0);

// 排序
const sorted = [...numbers].sort((a, b) => b - a); // 降序

// 反转
const reversed = [...numbers].reverse();

// 连接
const joined = numbers.join(","); // "1,2,3,4,5"

// 扁平化
const nested = [[1, 2], [3, 4]];
const flat = nested.flat(); // [1, 2, 3, 4]
```

## 元组（Tuple）

固定长度和类型的数组：

```ts
// 基本元组
const point: [number, number] = [10, 20];

// 带标签的元组
const user: [name: string, age: number] = ["Tom", 30];

// 可选元素
const optional: [string, number?] = ["hello"];

// 剩余元素
const rest: [number, ...string[]] = [1, "a", "b"];

// 只读元组
const readonly: readonly [string, number] = ["fixed", 123];
```

## Map

键值对集合：

```ts
// 创建
const map = new Map<string, number>();

// 设置
map.set("one", 1);
map.set("two", 2);

// 获取
map.get("one"); // 1
map.get("three"); // undefined

// 检查
map.has("one"); // true

// 删除
map.delete("one");
map.clear(); // 清空

// 大小
map.size;

// 遍历
map.forEach((value, key) => console.log(key, value));
for (const [key, value] of map) {
  console.log(key, value);
}

// 转换
const obj = Object.fromEntries(map);
const fromObj = new Map(Object.entries({ a: 1, b: 2 }));
```

## Set

唯一值集合：

```ts
// 创建
const set = new Set<number>();

// 添加
set.add(1);
set.add(2);
set.add(1); // 重复值被忽略

// 检查
set.has(1); // true

// 删除
set.delete(1);
set.clear();

// 大小
set.size;

// 遍历
set.forEach(value => console.log(value));
for (const value of set) {
  console.log(value);
}

// 数组去重
const unique = [...new Set([1, 1, 2, 2, 3])]; // [1, 2, 3]

// 集合操作
const a = new Set([1, 2, 3]);
const b = new Set([2, 3, 4]);

const union = new Set([...a, ...b]); // 并集 {1, 2, 3, 4}
const intersection = new Set([...a].filter(x => b.has(x))); // 交集 {2, 3}
const difference = new Set([...a].filter(x => !b.has(x))); // 差集 {1}
```

## 迭代器（Iterable）

### for...of

```ts
const arr = [1, 2, 3];

for (const item of arr) {
  console.log(item);
}

// 解构
for (const [key, value] of map) {
  console.log(key, value);
}
```

### 迭代器协议

```ts
function* numberGenerator(): Generator<number> {
  yield 1;
  yield 2;
  yield 3;
}

for (const num of numberGenerator()) {
  console.log(num);
}

// 展开生成器
const numbers = [...numberGenerator()]; // [1, 2, 3]
```

## 类型化数组

用于二进制数据：

```ts
// Int8Array
const int8 = new Int8Array(4);
int8[0] = 127;

// Uint8Array
const uint8 = new Uint8Array([1, 2, 3, 4]);

// Float64Array
const float64 = new Float64Array([1.1, 2.2, 3.3]);

// ArrayBuffer
const buffer = new ArrayBuffer(16);
const view = new Int32Array(buffer);
```

## 与 Java/C# 的对比

| 特性 | Java | C# | TypeScript |
|------|------|-----|------------|
| 动态数组 | ArrayList | List<T> | Array |
| 链表 | LinkedList | LinkedList<T> | 无内置 |
| Map | HashMap<K,V> | Dictionary<K,V> | Map<K,V> |
| Set | HashSet<T> | HashSet<T> | Set<T> |
| 元组 | 无内置 | Tuple<T1,T2> | [T1, T2] |
| LINQ/Stream | Stream API | LINQ | 数组方法链 |

**Java Stream API**：
```java
List<Integer> result = numbers.stream()
    .filter(n -> n > 2)
    .map(n -> n * 2)
    .collect(Collectors.toList());
```

**TypeScript 等价**：
```ts
const result = numbers
    .filter(n => n > 2)
    .map(n => n * 2);
```

# 模块

TypeScript 使用 ES 模块系统，与 Java 的包系统和 C# 的命名空间有所不同。

## 导出（export）

### 命名导出

```ts
// utils.ts

// 导出变量
export const PI = 3.14159;

// 导出函数
export function add(a: number, b: number): number {
  return a + b;
}

// 导出类
export class Calculator {
  add(a: number, b: number) { return a + b; }
}

// 导出接口
export interface User {
  id: number;
  name: string;
}

// 导出类型别名
export type ID = string | number;

// 导出常量
export const { name, age } = user;
export const [first, second] = array;

// 重命名导出
export { add as sum };
```

### 默认导出

```ts
// User.ts
export default class User {
  constructor(public name: string) {}
}

// 每个模块只能有一个默认导出
```

### 重新导出

```ts
// 导出其他模块的内容
export { add, subtract } from "./math";

// 重命名后导出
export { add as sum } from "./math";

// 导出全部
export * from "./utils";

// 导出为命名空间
export * as utils from "./utils";
```

## 导入（import）

### 命名导入

```ts
import { add, PI, Calculator } from "./utils";

// 重命名导入
import { add as sum } from "./utils";

// 导入全部
import * as utils from "./utils";
utils.add(1, 2);
```

### 默认导入

```ts
import User from "./User";

// 混合导入
import User, { Utils } from "./module";
```

### 动态导入

```ts
async function loadModule() {
  const { add } = await import("./math");
  add(1, 2);
}
```

### 类型导入

```ts
// 仅导入类型（编译后被移除）
import type { User } from "./types";

// 组合导入
import { User, type UserRole } from "./types";
```

## 模块解析

### 相对路径

```ts
import { something } from "./utils";    // 同目录
import { something } from "../parent";  // 父目录
import { something } from "./sub/file"; // 子目录
```

### 非相对路径

```ts
import { something } from "lodash";     // node_modules
import { something } from "@utils/api"; // 路径别名
```

### 路径别名配置

```json
// tsconfig.json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["src/*"],
      "@utils/*": ["src/utils/*"],
      "@components/*": ["src/components/*"]
    }
  }
}
```

```ts
// 使用别名
import { api } from "@utils/api";
import { Button } from "@components/Button";
```

## 模块结构

### 推荐的项目结构

```
src/
├── index.ts           # 入口文件
├── types/
│   ├── index.ts       # 类型导出
│   └── user.ts        # 类型定义
├── utils/
│   ├── index.ts       # 工具导出
│   ├── math.ts
│   └── string.ts
├── services/
│   ├── index.ts
│   └── api.ts
└── models/
    ├── index.ts
    └── User.ts
```

### barrel 模式

```ts
// utils/index.ts
export * from "./math";
export * from "./string";
export * from "./array";

// 使用时
import { add, capitalize, unique } from "./utils";
```

## CommonJS 兼容

```ts
// 导入 CommonJS 模块
import express from "express";
import * as fs from "fs";
import { readFileSync } from "fs";

// 导出为 CommonJS
export = {
  add,
  subtract
};
```

## 与 Java/C# 的对比

| 特性 | Java | C# | TypeScript |
|------|------|-----|------------|
| 模块定义 | package | namespace | 文件即模块 |
| 导入 | import | using | import |
| 访问控制 | public/private/protected | public/private/protected/internal | 无（所有导出项公开） |
| 命名空间 | package 层级 | namespace | 文件路径 |
| 别名 | import x.y.Z as A | using A = x.y.Z | import { Z as A } |

**Java 示例**：
```java
package com.example.utils;

public class Calculator {
    public static int add(int a, int b) {
        return a + b;
    }
}

// 使用
import com.example.utils.Calculator;
```

**TypeScript 等价**：
```ts
// utils/Calculator.ts
export function add(a: number, b: number): number {
  return a + b;
}

// 使用
import { add } from "./utils/Calculator";
```

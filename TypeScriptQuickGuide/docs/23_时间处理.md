# 时间处理

TypeScript/JavaScript 的时间处理主要依赖内置的 Date 对象。对于复杂的日期操作，推荐使用第三方库。

## Date 对象

### 创建日期

```ts
// 当前时间
const now = new Date();

// 指定时间戳
const timestamp = new Date(1672531200000);

// 指定日期字符串
const fromString = new Date("2024-01-01");
const fromIsoString = new Date("2024-01-01T10:30:00Z");

// 指定年月日时分秒
const specified = new Date(2024, 0, 1, 10, 30, 0); // 月份从0开始

// 只指定日期
const dateOnly = new Date(2024, 0, 1);
```

### 获取日期信息

```ts
const date = new Date();

date.getFullYear();     // 年
date.getMonth();        // 月 (0-11)
date.getDate();         // 日 (1-31)
date.getDay();          // 星期 (0-6, 0是周日)
date.getHours();        // 时
date.getMinutes();      // 分
date.getSeconds();      // 秒
date.getMilliseconds(); // 毫秒
date.getTime();         // 时间戳
date.getTimezoneOffset(); // 时区偏移（分钟）
```

### 设置日期

```ts
const date = new Date();

date.setFullYear(2025);
date.setMonth(11);      // 12月
date.setDate(25);
date.setHours(12);
date.setMinutes(0);
date.setSeconds(0);
```

### 格式化

```ts
const date = new Date();

// 内置格式化
date.toString();        // "Mon Jan 01 2024 10:30:00 GMT+0800"
date.toDateString();    // "Mon Jan 01 2024"
date.toTimeString();    // "10:30:00 GMT+0800"
date.toISOString();     // "2024-01-01T02:30:00.000Z"
date.toJSON();          // "2024-01-01T02:30:00.000Z"
date.toLocaleDateString(); // "2024/1/1"
date.toLocaleTimeString(); // "10:30:00"
date.toLocaleString();  // "2024/1/1 10:30:00"
```

### Intl.DateTimeFormat

```ts
const date = new Date();

// 简单格式化
new Intl.DateTimeFormat("zh-CN").format(date); // "2024/1/1"

// 详细格式化
new Intl.DateTimeFormat("zh-CN", {
  year: "numeric",
  month: "long",
  day: "numeric",
  weekday: "long",
  hour: "2-digit",
  minute: "2-digit"
}).format(date); // "2024年1月1日星期一 10:30"

// 相对时间
new Intl.RelativeTimeFormat("zh-CN").format(-1, "day"); // "1天前"
```

## 常用操作

### 日期计算

```ts
// 加减天数
function addDays(date: Date, days: number): Date {
  const result = new Date(date);
  result.setDate(result.getDate() + days);
  return result;
}

// 计算差值
function diffDays(date1: Date, date2: Date): number {
  const diff = date1.getTime() - date2.getTime();
  return Math.floor(diff / (1000 * 60 * 60 * 24));
}

// 获取月份天数
function getDaysInMonth(year: number, month: number): number {
  return new Date(year, month + 1, 0).getDate();
}
```

### 日期比较

```ts
const date1 = new Date("2024-01-01");
const date2 = new Date("2024-01-02");

// 比较（转换为时间戳）
date1.getTime() < date2.getTime();
date1.getTime() === date2.getTime();

// 判断是否同一天
function isSameDay(d1: Date, d2: Date): boolean {
  return d1.getFullYear() === d2.getFullYear() &&
         d1.getMonth() === d2.getMonth() &&
         d1.getDate() === d2.getDate();
}
```

## 第三方库推荐

### date-fns（推荐）

模块化、树摇优化的现代日期库：

```ts
import { format, addDays, differenceInDays, parseISO } from "date-fns";
import { zhCN } from "date-fns/locale";

// 格式化
format(new Date(), "yyyy-MM-dd HH:mm:ss"); // "2024-01-01 10:30:00"
format(new Date(), "yyyy年MM月dd日", { locale: zhCN });

// 日期计算
addDays(new Date(), 7); // 7天后
differenceInDays(date1, date2); // 天数差

// 解析
parseISO("2024-01-01T10:30:00Z");
```

### Day.js

轻量级（2KB），API 与 Moment.js 兼容：

```ts
import dayjs from "dayjs";

dayjs().format("YYYY-MM-DD HH:mm:ss");
dayjs().add(7, "day");
dayjs("2024-01-01").isValid();
```

### Luxon

现代化的日期时间库：

```ts
import { DateTime } from "luxon";

DateTime.now().toFormat("yyyy-MM-dd HH:mm:ss");
DateTime.now().plus({ days: 7 });
DateTime.fromISO("2024-01-01");
```

## 与 Java/C# 的对比

| 特性 | Java | C# | TypeScript |
|------|------|-----|------------|
| 日期时间类 | LocalDateTime | DateTime | Date |
| 日期类 | LocalDate | DateTime（仅日期） | 无内置 |
| 时间类 | LocalTime | TimeSpan | 无内置 |
| 时区 | ZonedDateTime | DateTimeOffset | Date（有限支持） |
| 格式化 | DateTimeFormatter | ToString(format) | toLocaleString() |
| 日期计算 | plusDays() | AddDays() | 手动实现或库 |
| 持续时间 | Duration | TimeSpan | 无内置 |

**Java 示例**：
```java
LocalDateTime now = LocalDateTime.now();
String formatted = now.format(DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss"));
LocalDateTime later = now.plusDays(7);
```

**TypeScript（date-fns）等价**：
```ts
import { format, addDays } from "date-fns";

const now = new Date();
const formatted = format(now, "yyyy-MM-dd HH:mm:ss");
const later = addDays(now, 7);
```
